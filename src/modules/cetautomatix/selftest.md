##### Selftest codes

| Code | Description                        |
|------|------------------------------------|
| 0x00 | Selftest initialized with success  |
| 0x10 | Testing Hardware                   |
| 0x20 | Testing Network                    |
| 0x30 | Testing I2C buses                  |
| 0x40 | Testing ROS modules                |
| 0x50 | Testing connection to other robots |
| 0x60 |                                    |
| 0xFF | Selftest completed successfuly     |
